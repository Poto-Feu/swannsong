# SwannSong
A text-adventure game with some humour

##Building the game

###Building prerequisites

####Linux
* glibc
* gcc
* cmake

####Windows
* mingw
* cmake


###Compiling

####Linux
```bash
mkdir build && cd build
cmake .. && cp -r ../txt .
make
```
