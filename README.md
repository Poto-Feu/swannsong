# SwannSong
A text-adventure game with some humour

## Building the game

### Building prerequisites

#### Unix-like (including Linux)
* glibc
* gcc
* cmake

#### Windows
* mingw
* cmake


### Compiling

#### Unix-like
```bash
mkdir build && cd build
cmake .. 
cp -r ../txt ../gameconf.txt .
make
```
